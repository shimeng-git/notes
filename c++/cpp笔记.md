## 基础用法

### 类静态数据成员

C++的非只读类内静态数据成员需要只能在类外（非头文件）初始化。

首先明确一点，C++中存在`定义`和`声明`两个概念。类内的静态数据成员实际上是声明，类外才是“定义”，因此需要在类外（源文件，最好不是头文件）添加对应的定义（初始化）语句才可以使用对应的静态数据成员。特殊的，对于const修饰的整形和枚举可以在类内完成定义，constexpr修饰的可以在类内完成定义（初始化）。

**静态数据成员不能在构造函数初始化：静态数据成员属于类而不是某个实例，如果类没有对应的实例化对象，也需要可以访问到对应的变量才行。**

```cpp
// Test.h
calss	Test {
private:
  static int cnt;	// ok, declare a variable
  static const int type = 0;	// ok, define. const 变量可以直接define（仅限于整型和枚举）
};
```
